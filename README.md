

## android平台音视频学习实践

![](https://github.com/byhook/ffmpeg4android/blob/master/readme/images/ffmpeg-logo.png)

## 环境配置

- 操作系统：`ubuntu 16.05`
- ndk版本：`android-ndk-r15c`
- ffmpeg版本：`3.3.8和3.4.6`
- fdk-aac版本：`0.1.6`
